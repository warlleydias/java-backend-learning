# Projeto de Orientação a Objetos (UnB)

Este repositório contém meus estudos de POO seguindo padrões de mercado para desenvolvimento Backend.

## 🚀 Tecnologias e Ferramentas
- **Java 21**: Uso de Records e novas funcionalidades da linguagem.
- **Maven**: Gestão de dependências e automação de build.
- **Lombok**: Redução de boilerplate (Getters/Setters/Constructors).
- **MapStruct**: Mapeamento performático de Entidades para DTOs.

## 🏗️ Arquitetura
O projeto está organizado em camadas para garantir o encapsulamento e a separação de responsabilidades:
- `model`: Entidades principais com regras de negócio.
- `dto`: Objetos de transferência de dados (Records).
- `mapper`: Interfaces de conversão automática.
- `exception`: Tratamento de erros personalizado.
